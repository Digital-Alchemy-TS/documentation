The `conversation` service contains utilities for interacting with conversation agents within Home Assistant.

| method              | description                           |
| ------------------- | ------------------------------------- |
| `addAlias`          | add an alias for a conversation agent |
| `removeAlias`       | remove alias from convesation agent   |
| `listAgents`        | list all known agents                 |
| `setConversational` | expose an agent as conversational     |
